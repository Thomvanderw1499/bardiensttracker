# Bardienst Tracker ⚽🍺

Bijhoud je bardiensten bij de voetbalclub. Exporteer elke 2 maanden naar Excel.

## Functies
- Snel bardiensten invoeren (datum, van-tot)
- Snelknoppen voor veelgebruikte tijden
- Achteraf bewerken of verwijderen
- Overzicht per 2-maandsperiode
- Export naar CSV/Excel met één klik
- Werkt offline als PWA

## Installeren via GitHub Pages

### Stap 1: Maak een GitHub repository
1. Ga naar [github.com](https://github.com) en log in
2. Klik op **New repository**
3. Geef het de naam `bardienst` (of wat je wil)
4. Zet op **Public**
5. Klik **Create repository**

### Stap 2: Upload de bestanden
Upload alle bestanden uit deze map naar je repository (via de webinterface of git).

### Stap 3: Zet GitHub Pages aan
1. Ga naar je repository → **Settings** → **Pages**
2. Bij **Source**: kies **GitHub Actions**
3. Push een wijziging of ga naar **Actions** → **Deploy Bardienst App** → **Run workflow**

### Stap 4: Installeer op Android
1. Open Chrome op je Android telefoon
2. Ga naar `https://JOUW-GEBRUIKERSNAAM.github.io/bardienst`
3. Chrome toont vanzelf een banner: **"App toevoegen aan startscherm"**
4. Tik op de banner of ga via het menu (⋮) → **App installeren**

De app werkt nu als een echte app op je telefoon, ook offline! 🎉

## Gebruik
- **Invoer**: Voer datum en tijden in. Gebruik de snelknoppen voor veelgebruikte tijden.
- **Overzicht**: Bekijk en bewerk diensten. Filter op periode.
- **Export**: Download CSV-bestand voor je Excel-overzicht.
- **Instellingen**: Stel clubnaam, naam en periodestart in.

## Privacy
Alle data wordt lokaal op je telefoon opgeslagen (localStorage). Er wordt niets naar een server gestuurd.
